# Portfolio V1

A modern, dark-themed portfolio website built with React and Vite. Featuring an Obsidian-inspired minimalist design with focus on Cybersecurity, AI, and DevOps projects.

## Features

- 🎨 Dark Obsidian-inspired theme
- 📱 Fully responsive design
- ⚡ Built with React and Vite
- 🎯 Smooth scrolling navigation
- 📝 Contact form with validation
- 🚀 Fast and lightweight

## Tech Stack

- React
- Vite
- CSS3 (CSS Variables for theming)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone git@github.com:frisatim/PortfolioV1.git
cd PortfolioV1
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
